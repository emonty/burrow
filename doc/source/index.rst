Welcome to Burrow's documentation!
**********************************

Burrow is a highly available, distributed, multi-tenant message
queue. Burrow can be used in a variety of environments, from simple
in-process queues to cloud-based message queue services.

This documentation is generated by the Sphinx toolkit and lives in the
source tree. Additional documentation on Swift and other components
of OpenStack can be found on the `OpenStack wiki`_.
